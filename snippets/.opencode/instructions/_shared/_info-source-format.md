## 情報源の通知フォーマット（必須）

Webで検索した情報を提案・報告に使用した場合、必ず通知する：

```
**参照情報源**
  - [情報源名（組織名）]：[URL]（取得日：[YYYY-MM-DD]）

**注意** 誤りや古い内容があればご指摘ください。
   指摘は decisions/ に記録し、以降の提案に反映します。
```

Web検索で裏付けが取れなかった場合：

```
**参照情報源**
  - AIの学習データに基づく提案（Web検索で裏付けが取れませんでした）
  - 確認を推奨：[公式サイト名と組織名]

**注意** 情報の鮮度に限界がある可能性があります。
```
