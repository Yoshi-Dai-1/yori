# CLI First

AI エージェントが外部リソース（コード・環境・データ・CI状態等）にアクセスする際は、常に CLI 経由を最優先する。このルールは全セッション・全フェーズで有効。

## 判断基準

- 「まず人間に聞く」ではなく「まず CLI を試す」がデフォルト動作
- CLI が未サポートの場合のみ、人間に支援を依頼する

## 手順

1. `command -v <tool>` で CLI の存在確認
2. `<tool> --help` で使い方を確認
3. 不足があれば `webfetch` で公式ドキュメントを調査
4. CLI が未サポートの場合のみ、人間に依頼

## 適用範囲

- コードの取得・確認（Git, clasp, gh, curl 等）
- 環境のセットアップ
- データの確認・取得
- デプロイ・CI/CD 状態の確認

上記は例示であり、未記載のサービス・ツールにもこの手順を適用する。
